# API Reference

Full reference for all public modules in `evalwire`.

| Module | Description |
|---|---|

| [uploader](uploader.md) | Upload CSV testsets to Phoenix |
| [runner](runner.md) | Discover and run experiments |
| [observability](observability.md) | OpenTelemetry tracing setup |
| [config](config.md) | Load and access `evalwire.toml` |
| [cli](cli.md) | `evalwire upload` / `evalwire run` CLI |
| [langgraph](langgraph.md) | LangGraph integration helpers |
